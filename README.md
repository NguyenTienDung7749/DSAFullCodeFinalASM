# DSA Full Code Final ASM

Đây là kho lưu trữ chứa mã nguồn Java cho bài tập cuối môn học Cấu trúc Dữ liệu và Thuật toán (DSA).

## Cấu trúc dự án

- **ASM1Code**: Mã cho bài tập 1
  - CodeThuyetTrinh: Mã cho phần thuyết trình
  - D2Code_ImperativeADT_OOP: Mã cho Ngày 2 - ADT Imperative và OOP
  - DataStructure: Cấu trúc dữ liệu
    - ArrayQueue
    - ArrayStack
    - LinkedList
    - LinkedListQueue
    - LinkedListStack
  - M2CodeTwoSortingAlgorithm: Mã cho Module 2 - Hai thuật toán sắp xếp
  - M3CodeUsingADT: Mã cho Module 3 - Sử dụng ADT
  - P2CodeMemoryStack: Mã cho Phần 2 - Bộ nhớ Stack

- **ASM2Code**: Mã cho bài tập 2
  - P4_P5Code: Mã cho Phần 4 và 5
  - P6CodeAsymptoticExamples: Mã cho Phần 6 - Ví dụ Asymptotic
  - P7CodeAlgorithmEfficiency: Mã cho Phần 7 - Hiệu quả thuật toán

## Cách chạy

1. Clone kho lưu trữ: `git clone https://github.com/NguyenTienDung7749/DSAFullCodeFinalASM.git`
2. Mở dự án trong IntelliJ IDEA (do có file .iml)
3. Chạy các lớp main tương ứng trong mỗi module.
